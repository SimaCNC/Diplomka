# Kalibrace snímačů malých posunutí

Tento projekt je řešen v rámci diplomové práce, slouží pro kalibraci snímačů malých posunutí, řádově v rozsahu 0 až 20 mm s rozlišením až desítek nanometrů. Kalibrace je založena na prováděna velmi malých translačních posunutí pomocí piezo aktuátorů s extrémně vysokým rozlišením pohybu (desítky nanometrů).
